Data Visualization in Practice

Chapter 4 Summary

In this brief three-day course, we provide a survey of the up-to-date data visualization methods and technologies. We have covered:

  • Basics of data visualization
  • Building charts by functions (and data types)
  • Generating interactive data products

Given the nature and time constraint of this short course, we cannot go into great details of each module, in particular the third one introducing new interactive data products. It is recommended students pursue more studies of these methods and start building own online gallery to present these data products. Data visualization goes beyond printed media and must consider online venues for presentation.
